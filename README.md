# DMap

[![Build Status](https://github.com/campbellmbrown/dmap-net/actions/workflows/ci.yml/badge.svg)](https://github.com/campbellmbrown/dmap-net/actions/workflows/ci.yml)
[![Documentation Status](https://github.com/campbellmbrown/dmap-net/actions/workflows/documentation.yml/badge.svg)](https://github.com/campbellmbrown/dmap-net/actions/workflows/documentation.yml)

A cross-platform desktop tool for running RPG map sessions. The DM loads a map image and reveals parts of it to players in real time over the local network.

See the [documentation](https://campbellmbrown.github.io/dmap-net/).
